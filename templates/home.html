  • Pick Manual
  • Set Primary DNS to {{ ui.highlight('147.135.213.57', '33,147,152') }}
  • and the Secondary DNS to {{ ui.highlight('1.1.1.1', '33,147,152') }}
  • Continue the setup according to your liking
  • Testing connection is optional
  • You should be done

With it, you can create a custom ship skin and bring it online!
To create a custom WipEout Pulse ship skin:

  • Download Thomas Perl's (aka thp) shipedit utility to create your skin, available for Linux and Windows
  • Export your custom skin
  • Send @chacheechoo the following information in Discord DM's:
    • The "16034453" file in the exported skin's folder;
    • Your PSN/PPSSPP Username;
    • The slot that you want to use (0-4);
    • The name of your custom skin;
    • Whether or not you would like the skin to be potentially publicly available to other players in the future
    • Note that your skin may take a while to be patched, or we could refuse to patch it
  • You will be given a patched skin - unpack the archive and put the UCES00465DTEAMSKIN000X folder in [PSP storage root]/PSP/SAVEDATA/
  • Check that the [Multiplayer...] -> [Online Account] -> [Ship Skins] option is on
  • You should be done

    • We are not responsible for any issues that may arise when using custom skins on any platform
